* Formatted Output - Printf
* Data Types in Go

## Prerequisites
* **Clone repository ini menggunakan command berikut (git bash)**
```
$ git clone https://github.com/sivaren/go-booking-app.git
```

## Run Program
* **Open `cmd` on this folder**
```
go run ./
```
